node manager => esgf shell... (esgf-sh)

The esgf shell provides a simple shell and command line interface to the node's services. Users can use the shell to perform any of the tasks that may be performed through the web user interface. To help end users leverage the shell to it's maximum the shell provides the ability for end-users to develop custom shell commands. Commands can be written in Java, Closure, Scala, Jython, Groovy, or BeanShell. Through custom commands end-users have a direct mechanism for extending the node features and creating new ones!
